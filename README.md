# ⚙️ Makine Hesaplamaları

Makine mühendisliği ve imalat süreçlerinde kullanılan temel hesaplamaları kolaylaştırmak amacıyla geliştirilmiş bir web uygulamasıdır.  
Bu proje, özellikle teknik çizim, CNC ve üretim süreçlerinde ihtiyaç duyulan hesapları hızlı ve pratik şekilde yapmayı hedefler.

---

## 🚀 Özellikler

- 📐 Temel makine hesaplamaları
- ⚡ Hızlı ve kullanıcı dostu arayüz
- 📱 Responsive (mobil uyumlu) tasarım
- 🧮 Anlık sonuç hesaplama
- 🎯 Pratik kullanım (karmaşık formülleri otomatik çözer)

---

## 🛠️ Kullanılan Teknolojiler

- HTML5  
- CSS3  
- JavaScript
